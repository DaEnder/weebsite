Who is P3tray?

And what has he been developing?

P3tray.TK

P3tray.TK (This website) is an open-source static webpage built with HTML and CSS. It was built without JavaScript, SASS, or any libraries. The images on the site are SVG's (Scaleable Vector Graphics) made by myself in Inkscape. It's currently hosted on GitHub with this custom domain name.

Lithium for Discord

Lithium for Discord is an open-source Discord bot built to provide helpful features to Discord owners, providing systems such as Suggestions. It also boasts systems such as the ability to write to game DataStores and run commands on game servers (Through Lithium Admin) all within Discord.

Lithium Admin

Lithium Admin is a closed-source command suite and anti-exploit for the Roblox game platform. It provides game owners and players sets of commands useful to host events, deal with abuse, and have fun. It is based of Adonis by Davey_Bones and follows it's update schedule.
